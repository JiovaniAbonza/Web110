Web Development Glossary

Glossary

HTML
HTML is a markup language for the web that defines the structure of web pages.
CSS
CSS is a language that lets you control how HTML elements look in the browser.
JavaScript
JavaScript is the programming language for the web that can update and change HTML, CSS, and data.
Responsive Design
Responsive Design is where you can make the web page fit on many other devices.
Flexbox
Flexbox is a layout that can help you make rows or columns which ever prefered.
Grid Layout
Grid Layout is a two-dimensional layout system for the web.
